News delivery

O News Delivery é um aplicativo móvel para gerenciamento de entregas de jornais, que permite automatizar o controle das entregas, notificar clientes e definir rotas para os entregadores. O entregador pode acessar sua rota por meio de um mapa, permitindo que o sistema notifique o cliente automaticamente quando o entregador se aproxima a 100 metros do endereço de entrega. As informações referentes às entregas são salvas em um banco de dados para gerar relatórios com dados importantes. O aplicativo é disponibilizado apenas para administradores e funcionários do jornal.

Supermercado online

O Supermercado Online é um sistema similar a um e-commerce, porém adaptado para atender às necessidades de supermercados. Esse sistema apresenta funcionalidades abrangentes que contemplam todas as etapas do processo de compra, desde a seleção dos produtos até a entrega dos mesmos. O usuário tem a capacidade de acompanhar o status de seu pedido e receber notificações quando o pedido está prestes a ser entregue.
